### 4. 构建修复
- 首次 `npm run build` 失败：`MenuBarMockup.astro` 第 114 行附近 `Expected ")" but found "$$render"`。
- 根因：Astro 不允许在 `{variant === "hero" && ( {/* comment */} <div>... )}` 这种 `&&` 表达式后直接跟注释表达式块；且原结构把 hero mockup 作为 dropdown 的兄弟节点，逻辑反了。
- 修复：重写组件，用 `{variant === "hero" ? (...) : null}` 三元表达式替代 `&&`，移除条件渲染内的行内注释；hero variant 内联完整 dropdown（不再用 `<slot />`），保证输出是「菜单栏 + dropdown 浮层」完整组合。
- 修复后 `npm run build` 成功，产出 `dist/`（约 76KB，1 页）。
