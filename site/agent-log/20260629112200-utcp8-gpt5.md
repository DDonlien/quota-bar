## 对话与行动记录

### 1. 需求解读

用户截图（reference 风格）里 Hero 标题是 "Never lose track of your AI quotas"。要求把 "AI" 换成**动态打字机效果**：
- 删除当前 → 重新打字
- 轮播的词 = Quota Bar 实际支持的 agent：`Antigravity / Codex / Claude / MiniMax / Kimi / Zcode`（共 6 个，从 `Layout.astro` description 提取）
- 新打出来的文字 = **橙色**

### 2. 实现思路

- 在 Hero.astro 的 H1 里把 `AI` 替换成 `<span data-typewriter="..." class="typing-text">`，数据用 `|` 分隔给前端解析
- 加本地 `<style>` 定义橙色渐变（不能用 Layout 的 `.text-gradient`，因为 Astro scoped style 不会传到 child component）
- 加 `<script is:inline>` 实现打字机循环（is:inline 防止 Astro 改写 script 路径，参考上一轮 Tailwind CDN 修复）
- 加 `prefers-reduced-motion` 降级

### 3. 关键代码点

**frontmatter**：
```js
const AGENTS = ["Antigravity", "Codex", "Claude", "MiniMax", "Kimi", "Zcode"];
const AGENTS_TYPED = AGENTS.join("|");
```

**H1 改动**：
```diff
- <h1 class="text-5xl md:text-6xl leading-tight mb-6 font-bold">
-   Never lose track of your AI quotas
- </h1>
+ <h1 class="text-5xl md:text-6xl leading-tight mb-6 font-bold">
+   Never lose track of your
+   <span class="typing-text" data-typewriter={AGENTS_TYPED} aria-live="polite">Antigravity</span>
+   quotas
+ </h1>
```

**本地 style**（橙色渐变 + 闪烁光标）：
```css
.typing-text {
  display: inline-block;
  background: linear-gradient(90deg, #f59e0b 0%, #d97706 100%);
  -webkit-background-clip: text;
  background-clip: text;
  -webkit-text-fill-color: transparent;
  color: transparent;
  vertical-align: baseline;
  min-width: 1ch;
}
.typing-text::after {
  content: "|";
  color: #f59e0b;
  -webkit-text-fill-color: #f59e0b;
  background: none;
  animation: typing-cursor 1s steps(1) infinite;
}
@keyframes typing-cursor {
  0%, 50% { opacity: 1; }
  50.01%, 100% { opacity: 0; }
}
```

**打字机 JS**（ES5，老 Safari 兼容）：
- TYPING_SPEED = 90ms/char
- DELETING_SPEED = 45ms/char
- PAUSE_AFTER_TYPE = 1400ms
- PAUSE_AFTER_DELETE = 300ms
- 支持多个 `[data-typewriter]` 元素 + 防重复 init

### 4. 验证

用 Node + CDP（chrome remote debugging protocol）写了 12 行小脚本 `/tmp/cdp-screenshot.js`，在 0 / 500 / 1500 / 2500 / 4000 / 5500 / 8000ms 截 7 张图，同时通过 Runtime.evaluate 读 `data-typewriter` 的 textContent。

观察到的文字序列：
- 0ms: "Antigra"（打字中）
- 500ms: "Antigravity"（打完 + 光标）
- 1500ms: "Antigravity"（停留中）
- 2500ms: "Co"（已删完 Antigravity，开始打 Codex）
- 4000ms: "Codex"（打完 + 光标）
- 5500ms: "Claude"
- 8000ms: "MiniMax"

**结论**：✅ 打字机 + 删除 + 6 agent 轮播 + 橙色渐变 + 闪烁光标全部生效。
