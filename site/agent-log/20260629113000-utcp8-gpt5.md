## 对话与行动记录

### 1. 字体 CDN 调研

**JetBrains Mono**：
- GitHub 仓库 webfonts 目录 jsdelivr 直链 404（实际有文件但路径不对）
- jsdelivr npm 包可用：`https://cdn.jsdelivr.net/npm/jetbrains-mono@1.0.6/fonts/webfonts/JetBrainsMono-Regular.woff2` (200)
- **NL（No-Ligature）= OpenType feature flag `calt 0`，不是独立的 woff2 文件**
- 现代 JetBrains Mono 仓库里没有 NL 命名的 woff2

**Maple Mono**：
- 仓库 v7.9 release 提供多个 woff2 zip
- `MapleMonoNL-Woff2.zip` 1.1 MB — **Latin only，不带中文**
- `MapleMonoNL-CN.zip` 137 MB — 带中文但太大
- `MapleMonoNL-NF-CN.zip` 155 MB — 更大
- fontsource npm 包只有 latin CSS（无 chinese-simplified subset）
- deno-deploy CDN 重定向到失效 URL（403）

**决定**：
- 本地放 JetBrains Mono Regular + Bold woff2（按字符类型分字体，NL 走 CSS feature flag）
- 本地放 Maple Mono NL Regular woff2（Latin only）作为参考
- **接受中文 fallback 到系统字体（PingFang SC / 苹方 / 微软雅黑）** — 因为 CN woff2 太大，CSS 里不引 Maple Mono，中文字符由 system-ui 接管
- 后续如果 user 坚持中文也用 Maple Mono，需要 pyftsubset 做 CJK subset 切片

### 3. Layout.astro 字体方案

```css
@font-face {
  font-family: "JetBrains Mono";
  src: url("/fonts/jetbrains/JetBrainsMono-Regular.woff2") format("woff2");
  font-weight: 400;
  font-style: normal;
  font-display: swap;
  /* 拉丁基础 + 扩展 + 补充 + 标点 + 数字 + 箭头 + 数学符号 */
  unicode-range: U+0020-007F, U+00A0-00FF, U+0100-017F, ...;
  font-feature-settings: "calt" 0, "liga" 0;  /* NL = No-Ligature */
}
@font-face {
  font-family: "JetBrains Mono";
  src: url("/fonts/jetbrains/JetBrainsMono-Bold.woff2") format("woff2");
  font-weight: 700;
  font-feature-settings: "calt" 0, "liga" 0;
  /* 同 unicode-range */
}

body {
  font-family: "JetBrains Mono", -apple-system, BlinkMacSystemFont,
    "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", monospace;
}
```

**关键点**：
- 字体策略按**字符类型**分：拉丁字符 → JetBrains Mono（NL）；CJK 字符 → 系统字体
- unicode-range 让浏览器按字符自动选字体
- 移除 Inter / Press Start 2P 引用（之前是从 Google Fonts 拉的）
- 移除 `font-feature-settings: "ss01", "cv01"`（那是 Inter 的 feature）

### 5. Features.astro vibeisland 风格 + hover

**HTML 结构**：
- `<article class="feature-card">` 替代原来的 `<div class="glass-card">`
- `.feature-icon` / `.feature-title` / `.feature-desc` 三层语义结构
- 去掉 Tailwind 的 `text-center flex flex-col items-center`，改用 CSS 布局

**CSS 关键点**：
```css
.feature-card {
  display: flex; flex-direction: column; align-items: center; text-align: center;
  padding: 40px 32px;
  background: rgba(25, 25, 28, 0.4);
  border: 1px solid rgba(255, 255, 255, 0.06);
  border-radius: 16px;
  transition: transform 220ms cubic-bezier(0.16, 1, 0.3, 1),
              border-color 220ms ease, background 220ms ease, box-shadow 220ms ease;
}
.feature-card:hover {
  transform: translateY(-3px);
  background: rgba(30, 30, 34, 0.7);
  border-color: rgba(245, 158, 11, 0.55);   /* 橙，跟打字机高亮一致 */
  box-shadow: 0 12px 32px rgba(0, 0, 0, 0.4),
              0 0 0 1px rgba(245, 158, 11, 0.15);
}
.feature-icon {
  width: 44px; height: 44px;
  display: flex; align-items: center; justify-content: center;
  color: rgba(255, 255, 255, 0.5);
  background: rgba(255, 255, 255, 0.04);
  border: 1px solid rgba(255, 255, 255, 0.06);
  border-radius: 10px;
  margin-bottom: 20px;
  transition: color 220ms ease, background 220ms ease, border-color 220ms ease;
}
.feature-card:hover .feature-icon {
  color: #f59e0b;                            /* 橙 */
  background: rgba(245, 158, 11, 0.12);
  border-color: rgba(245, 158, 11, 0.4);
}
```

**响应式**：
- 1 列（< 768px）
- 2 列（768-1024px）
- 3 列（≥ 1024px）

**可访问性**：
- `@media (prefers-reduced-motion: reduce)` 关闭 transform / transition
