## 修复方案

### 1. 新建 SettingsComponents.swift

自定义 4 个组件：
- `SettingsSection("title") { ... }`：section title（13pt semibold primary）+ 8pt 间距
- `SettingsGroup { ... }`：10pt 圆角矩形 + `.regularMaterial` 背景 + 0.5pt separator 描边
- `SettingsRow(label:subtitle:trailing:)`：13pt label + 11pt caption subtitle + trailing
- `SettingsDivider`：从 row 左边距（16pt）开始的水平分割线

### 2. PreferencesScene 重写

- Sidebar：
  - `List(selection: $selection)` + `.listStyle(.sidebar)`
  - 每个 row 的 `Label` icon 加 `foregroundStyle(section.tint)` 彩色
  - 用 `Section("title") { ... }` 自动 header
  - 不手动染蓝（system 自动 glass 高亮）
- Detail：弃用 `Form`，用 `ScrollView` + `SettingsSection` + `SettingsGroup` + `SettingsRow`
- 整体加 `.background(.regularMaterial)`

### 3. 4 个 detail view 全部重写

- `GeneralSettingsView`：4 个 section（刷新 / 数据来源 / 菜单栏 / 启动），slider 拆成两行（label + value / slider + 范围）
- `ModelsSettingsView`：1 个 section "Provider" + 1 个 Group 包所有 provider（icon + name + subtitle + toggle）
- `ActivationSettingsView`：3 个 section（当前状态 / 本机 / 说明）
- `AboutSettingsView`：3 个 section（应用 / 链接 / 维护）

### 4. PreferencesWindowController 改 titlebar

- `titlebarAppearsTransparent = true` + `.fullSizeContentView`：traffic light 浮在 sidebar 顶部
- `titleVisibility = .hidden`：把 title 让给 SwiftUI NavigationSplitView 的 inline title
- `backgroundColor = .clear`：让 SwiftUI 端 `.regularMaterial` 玻璃背景透出
