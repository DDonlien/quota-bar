# 对话与行动记录

- 回应用户关于权限的疑问：Full Disk Access 只能解决读取浏览器 Cookie 数据库文件的问题；Chromium 系浏览器的 Cookie value 通常还需要 Keychain Safe Storage 解密，因此 FDA 不能单独替代浏览器/Keychain 授权。当前默认链路继续避免浏览器 Cookie。
- 调研 Kimi：确认 `~/.kimi-code` 的 CLI OAuth 只能覆盖 Code 5h / week；发现 Kimi Desktop `bridge-store/token-store.json` 存在 Web access token，适合作为 Work + Code 的默认非浏览器来源。
- 调研 Z Code：确认本地 `~/.zcode/v2/coding-plan-cache.json` 能反映 plan 是否 available / `coding_plan_not_entitled`，但不包含可渲染 quota 数值，因此只能展示真实状态，不能生成额度条。
- 尝试实网请求 Kimi membership API 时受执行环境网络/权限限制，未继续绕过；改用单元测试验证请求契约与 parser 结果。
