## 对话与行动记录

1. 按用户要求放弃继续模拟 macOS 26 dropdown，转向传统 macOS dropdown。
2. 下载参考仓库：
   - `reference/project/Mos`
   - `reference/project/hidden`
3. 阅读 Mos 与 Hidden Bar 的状态栏菜单实现：
   - Mos 使用 `NSStatusBar.system.statusItem` 并将 `item.menu = self`，菜单项使用 `NSMenuItem` 和系统分隔线。
   - Hidden Bar 为状态栏项目挂载 `NSMenu`，需要手动弹出时使用 `menu.popUp(positioning:at:in:)`。
4. 重写 `StatusBarController.swift`：
   - 移除自绘 `NSWindow`、`NSGlassEffectView`、全局点击监听和手动定位逻辑。
   - 使用 `NSStatusItem.menu` 挂载原生 `NSMenu`。
   - 菜单顶部嵌入透明 `NSHostingView(rootView: MenuView())` 作为信息 dashboard。
   - 底部操作改为原生 `NSMenuItem`：立即刷新、偏好设置、退出。
5. 调整 `MenuView.swift`：
   - 作为 `NSMenuItem.view` 内的透明 dashboard 使用。
   - 尺寸改为 `292×292`。
   - 字号缩小到传统菜单密度：顶部 13 pt、订阅 14 pt、额度 11 pt。
   - 移除底部自绘退出按钮。
6. 执行 `swift build`，通过。
7. 停止旧进程，复制新二进制到 `CodingPlanMenu.app/Contents/MacOS/CodingPlanMenu`，并用 `open CodingPlanMenu.app` 验证新进程启动，PID 为 `2257`。
8. 更新 `README.md`、`REQUIREMENTS.md` 与 `DESIGN.md`。
